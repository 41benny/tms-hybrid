# Fix cPanel Deployment Error

## Error Message
```
Error: (XID j4xusk) "/usr/local/cpanel/3rdparty/bin/git" reported error code "1" when it ended: 
error: Your local changes to the following files would be overwritten by merge: 
resources/views/cash-banks/print.blade.php 
Please commit your changes or stash them before you merge. Aborting
```

## Root Cause
Ada perubahan manual di server production pada file `print.blade.php` yang belum di-commit.

## Solution Options

### Option 1: Via cPanel Terminal (Recommended)
1. Login ke cPanel
2. Buka "Terminal" atau "SSH Access"
3. Navigate ke repository:
   ```bash
   cd ~/repositories/tms-hybrid
   ```
4. Stash local changes:
   ```bash
   git stash
   ```
5. Kembali ke Git Version Control dan klik "Deploy HEAD Commit"

### Option 2: Via File Manager
1. Login ke cPanel File Manager
2. Navigate ke: `/home/vintamal/repositories/tms-hybrid/resources/views/cash-banks/`
3. Backup file `print.blade.php` (download atau rename)
4. Delete file `print.blade.php`
5. Kembali ke Git Version Control dan klik "Deploy HEAD Commit"
6. File akan di-recreate dari repository

### Option 3: Force Reset (DESTRUCTIVE - will lose server changes)
Via SSH/Terminal:
```bash
cd ~/repositories/tms-hybrid
git reset --hard HEAD
git pull origin main
```

## Prevention
Untuk mencegah issue ini di masa depan:
- Jangan edit file langsung di server production
- Semua perubahan harus melalui Git workflow (local → commit → push → deploy)
